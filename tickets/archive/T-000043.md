## Goal
Allow users to explicitly save completed Human vs Stockfish games as PGN from Play tab.

## Acceptance Criteria
- [ ] Play tab includes explicit `Save Human PGN` action in Play controls.
- [ ] Save action exports the current completed Human vs Stockfish game with correct player names and result.
- [ ] Save action is disabled unless mode is Human and current game is completed.
- [ ] When user attempts save in invalid state, UI shows a clear status message instead of downloading.

## Goal
Add a Play-tab demo mode that continuously runs engine-vs-engine games until stopped.

## Acceptance Criteria
- [ ] Play controls include explicit `Start Demo Mode` action in engine mode.
- [ ] Clicking `Start Demo Mode` continuously runs engine-vs-engine games back-to-back until `Stop` is clicked.
- [ ] Engine status updates after each completed demo game with deterministic outcome text (white win / black win / draw / game over).
- [ ] Demo mode does not run in Human mode and respects existing busy-state control disabling.
